# mytube

Use Vue-cli + Youtube video-API to make a basic RWD project.

---

### How

> npm run serve


### Set youtube API-key

1. provide an .env file

> VUE_APP_VIDEO_KEY={$your_key}

2. OR fill in your key in the main page.
